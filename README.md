# canto-backstage

原本是想使用 vue-antd-pro 来作为脚手架 但是下载完项目之后发现里面的文件还是比较的多，一部分的功能是我不需要的
又不想一个个去删除，索性自己模仿了它的脚手架，抽取了其中的大部分功能做了一个简化版本的脚手架以便以后公司或者个人
需要快速搭建简易后台项目的时候使用。
